\doxysubsection{Descrição detalhada}
Declara classes utilitárias para carregar e usar imagens, sons e fontes no Allegro. 

Fornece wrappers RAII ({\ttfamily \mbox{\hyperlink{classImage}{Image}}}, {\ttfamily \mbox{\hyperlink{classSound}{Sound}}}, {\ttfamily \mbox{\hyperlink{classMusic}{Music}}}, {\ttfamily \mbox{\hyperlink{classTextFont}{Text\+Font}}}) e helpers para assets globais utilizados em vários estados da aplicação.
