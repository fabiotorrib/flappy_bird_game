Gerencia a lógica e a renderização da tela de seleção de dificuldade. 

Bibliotecas necessárias

Herda da classe base \textquotesingle{}\mbox{\hyperlink{classState}{State}}\textquotesingle{} e implementa o comportamento específico para permitir que o usuário escolha e salve um nível de dificuldade para o jogo.
