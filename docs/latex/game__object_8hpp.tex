\doxysubsection{Descrição detalhada}
Classe base abstrata para todos os objetos renderizáveis. 

Define posição, dimensões e interface para atualização/desenho. Todas as entidades que podem colidir ou ser renderizadas devem herdar desta classe e implementar os métodos puros {\ttfamily update()} e {\ttfamily draw()}.
